# NEXUS — Site estático (v1)

Este pacote contém um **site estático** pronto para publicar na **Vercel** (ou qualquer hospedagem de arquivos).

## Arquivos
- `index.html` — página única com todas as seções.
- `styles.css` — estilos básicos (sem dependências).
- Imagens via links externos (Unsplash).

## Como publicar na Vercel (resumo)
1. Crie um repositório no GitHub e envie estes arquivos (Upload via web).
2. Na Vercel: **Add New Project** → **Import Git Repository** → selecione o repo.
3. Framework: **Other** (ou Static) | Build Command: _vazio_ | Output Directory: `/`.
4. Deploy e pronto.

## Domínio
Adicione `nexusriopreto.com.br` e `www.nexusriopreto.com.br` no projeto (Settings → Domains) e siga as instruções de DNS.
